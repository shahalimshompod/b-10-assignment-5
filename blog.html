Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It is the structure of an HTML document as a tree of objects, where each elements in the document becomes a node of the tree. We can easily manipulate, design and make our website interactive through javascript.

How do you select an element from the DOM?

There are 6 ways to select an element from the DOM lets say- getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getElementsByName()... But I frequently use getElementById() and getElementsByClassName() to select element from the DOM. For getElementById(), first I put a id in which element I want to select. After that I put the id name in the ('') of getElementById(). For example getElementById('id-name')

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is a technology in the DOM where we don't need to attach event listeners to multiple child element. We can attach a single event listener to their parent element. Cause this works with the 'bubble up' technique. It is usefull cause is reduces the number of event listeners and improving performance. And It handles dynamic elements.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes and styles in the DOM using simple methods. To get an attribute from an element, I use element.getAttribute('attributeName');, and to set a new attribute I use element.setAttribute('attributeName', 'value');. If I want to remove an attribute, I do element.removeAttribute('attributeName');. For styles, I can modify an element's inline styles by accessing the style property. For example, to change the color of an element, I write element.style.color = 'red';. If I working with CSS classes, I add a class to an element using element.classList.add('className');, or remove it with element.classList.remove('className');.
